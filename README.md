# 🏡 Modern Real Estate Landing Page

A modern, responsive, and visually appealing Real Estate Landing Page built using HTML, CSS, and JavaScript. This project showcases premium properties, real estate services, and a professional user experience designed to attract buyers, renters, and investors.

👉 View Live Demo:(https://roghayem.github.io/real-estate-landing-page/)

## ✨ Features

* 🏡 Modern real estate website design
* 📱 Fully responsive on desktop, tablet, and mobile
* 🔍 Property search section
* 🏘️ Featured properties showcase
* 📸 High-quality property galleries
* 📍 Location and neighborhood information
* ⭐ Client testimonials section
* 📊 Property statistics and market insights
* 📞 Contact agents and inquiry forms
* 🚀 Smooth animations and interactions

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Flexbox
* CSS Grid
* Media Queries
